🔧 BACKEND - NUEVOS ENDPOINTS

POST /api/inscripciones
  → Alumna se inscribe a una clase
  → Requiere: clase_id
  → Devuelve: id de inscripción

DELETE /api/inscripciones/:claseId
  → Alumna cancela una clase
  → Solo cancela SU inscripción
  → La clase sigue disponible para otros
  → Devuelve: mensaje de confirmación
